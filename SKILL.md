# 南京审计大学学术讲座爬虫技能

## 概述

本技能从南京审计大学官网学术活动栏目爬取讲座信息，支持状态智能判定（未开始 / 已过时 / 未知），终端交互为主，按需生成自包含静态 HTML 可视化。

## 触发条件

当用户表达以下意图时使用此 skill：
- "查看南京审计大学讲座"
- "查一下 NAU 最近的讲座"
- "获取南审学术讲座"
- 任何涉及南京审计大学讲座查询的需求

## 使用流程

### 步骤一：爬取数据 → 终端交互

```bash
python scripts/crawler.py
```

终端打印完整讲座摘要表格 + 链接 + 交互提示。LLM 读取数据后与用户对话：分析讲座内容、推荐合适的场次、回答用户问题。

**此时不生成 HTML，纯终端交互。**

### 步骤二：按需生成可视化面板

LLM 在给出建议后**必须**追问一句：

> 需要我给出可视化面板吗？

- 用户确认 → 运行 `python scripts/crawler.py -m` 生成 HTML 并打开
- 用户拒绝 → 继续终端交互

### 响应模板

**步骤一输出：**

```
──────────────────────────────────────────────────────
  ● 未开始 2026-05-06 14:30  传统文化视野中的苏东坡…  [敏达学术报告厅一楼]
  ● 未开始 2026-05-13 14:00  供应链地理分布扩张…      [敏知516会议室]
  O 已过时  …                 …
──────────────────────────────────────────────────────

  共 N 条讲座（未开始 X / 已过时 X / 未知 X）
  [WEB] 网页查看: file:///.../lectures.html

  [TIP] 你有哪些想听的讲座？需要我给出可视化面板吗？
```

**步骤二触发：**

```bash
python scripts/crawler.py -m
```

### 参数选项

```bash
python scripts/crawler.py -m out.html     # 指定 HTML 文件名
python scripts/crawler.py -m out.html --fast  # 快速模式（不爬详情）
python scripts/crawler.py -o data.json    # 仅输出 JSON 到文件
python scripts/crawler.py -q              # 静默模式
```

## 技能结构

```
nau-lecture-skill/
├── SKILL.md                   # 本文件
├── scripts/
│   └── crawler.py             # 数据获取 + HTML 生成
└── references/
    └── template.html           # 单文件多主题可视化模板（CSS变量切换）
```

## HTML 界面功能

| 功能 | 说明 |
|------|------|
| 多主题切换 | 点击按钮循环切换 6 种配色（墨砚/深海/赤霞/竹韵/暮紫/霜白），CSS 变量驱动 |
| 统计卡片 | 总讲座数、未开始 / 已过时 / 未知数量 |
| 状态筛选 | 一键过滤三种状态（默认筛选"未开始"） |
| 关键词搜索 | 实时搜索标题、报告人、主办方 |
| 时间排序 | 点击表头切换升降序（▲/▼ 指示方向） |
| 相对时间 | 时间下方显示"3天后"、"明天"等提示 |
| 颜色标记 | 行底色按状态自动着色（绿 / 红 / 灰） |
| 日志面板 | 折叠式，点击标题展开 |
| 详情链接 | 点击直接打开原讲座页面 |

## 状态判定规则

| 状态 | 颜色 | 判定逻辑 |
|------|------|----------|
| 未开始 | 绿 `#00B42A` | 当前时间 < 讲座时间 |
| 已过时 | 红 `#F53F3F` | 当前时间 >= 讲座时间 |
| 未知 | 灰 `#86909C` | 无法解析讲座时间（兜底，提示官网格式变动） |

## 扩展与修改

### 修改爬取目标
编辑 `scripts/crawler.py` 的 `MAIN_URL` 变量。

### 添加时间格式
在 `format_time()` 函数（约第 53 行）的 `patterns` 列表中添加正则匹配规则。

### 自定义样式
修改 `references/template.html` 中各主题的 CSS 变量。
